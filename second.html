Show the Scope of variables in JS using HTML page.

JavaScript has 3 types of scope:

1. Block Scope
2. Function Scope
3. Global Scope

We will use the same variable x to demonstrate the scope of variables in JS

1. Block Scope

ES6 introduced two important new JavaScript keywords: let and const.

These two keywords provide Block Scope in JavaScript.

Variables declared inside a { } block cannot be accessed from outside the block:

Example:
Here we can use the varible x only in the {}

{ let x = 5; function scope(){ alert("Value of variable x in the block: " + x) } } // x cannot be used here

2. Function Scope

JavaScript has function scope: Each function creates a new scope.

Variables defined inside a function are not accessible (visible) from outside the function.

Variables declared with var, let and const are quite similar when declared inside a function.

3. Global Scope

Variables declared Globally (outside any function) have Global Scope.

Global variables can be accessed from anywhere in a JavaScript program.

Variables declared with var, let and const are quite similar when declared outside a block.

Example:

We will use the variable x to demonstrate global and function scope

var x = 10; // x defined globally function scopeFunction(){ var x = 20; alert("Value of variable x in the function: " + x) } function scopeGlobal(){ alert("Value of variable x globally: " + x) }
